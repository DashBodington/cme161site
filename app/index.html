Visualization of All-Star Player and Team Statistics with Crossfilter

A visualization of baseball teams and players represented in the All-Star game using Crossfilter 1.3.12. Crossfilter allows for fast filtering of data along multiple dimensions, d3 is used to produce plots, and bootstrap for slider functionality. Data from Sean Lahman.

Visualization of Iris Properties using dc.js

This visualization shows various properties of irises, and allows subset selection through brushing and linking with dc.js 1.7.5. dc.js is a plotting library which encorporates crossfilter and graphics which allow for easy user interaction. Click and drag on any of the bar charts, or select slices from the pie chart.

Visualizing Donut Parameters with d3.js

This visualization represents the many options for crafting donuts in interactive tree format using d3.js 3.5.17. D3 is a library for visualizing data in the document object model in many ways. Click nodes to expand/shrink branches.

Visualizing Doping in Professional Cycling with c3.js

This visualization is a set of slides about doping in professional cycling. Data was collected from annual World Anti-Doping Agency reports here, and was visualized with c3.js 0.4.11. c3.js is a javascript library for creating charts. Click through the slides to view some of its capabilities.
